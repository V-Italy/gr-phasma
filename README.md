## Installation
To install the gr-phasma, the following dependencies are required:
- [GNU Radio](https://github.com/gnuradio/gnuradio)


Install by the following shell commands:
```
git clone git@github.com:ctriant/gr-phasma.git
cd gr-phasma
mkdir build
cd build
cmake ..
make -j4
sudo make install
```
